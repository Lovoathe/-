| 类别    | 物质：              |      |     |     |     |             |     |             |
| ----- | ---------------- | ---- | --- | --- | --- | ----------- | --- | ----------- |
| 盐     | $\ce{NH4+}$      |      |     |     |     | $\ce{NO2-}$ |     | $\ce{NO3-}$ |
| 酸/碱   | $\ce{NH3 * H2O}$ |      |     |     |     | HNO2        |     | HNO3        |
| 单质    |                  |      | N2  |     |     |             |     |             |
| 氢/氧化物 | $NH_3$           | N2H4 |     | N2O | NO  | N2O3        | NO2 | N2O5        |
| N     | -3               | -2   | 0   | +1  | +2  | +3          | +4  | +5          |

- 纵向不变价、横向变价（酸碱反应模型）
- N2的稳定是指即容易生成，又难分解
	- 一般 $NH_3$ 变价的反应，如果没有催化剂，则跳不过 N2（即生成氮气）
		- 比如 $NH_3$ +NO→N2
		- $NH_3$ +Cl2→N2+HCl
		- $NH_3$ +CuO→（加热）N2+Cu
		- $NH_3$ + $O_2$ →（点燃）N2
