# 羟氨
[[羟氨的性质]]
## 基础
- 氨气中的一个 H 被-OH 代替
- HO-NH2
## 性质
- 碱性（因为有孤对电子）
	- $\ce{HO * NH2 + H2O <=> [HO * NH3]+ + OH-}$
- 还原性
# 连氨
## 基本
- NH2-NH2（相当于氨气中有一个 H 被氨基取代）
- 电子式
```
   H    H
   ..   ..
H : N : N : H
   ..   ..
   H    H
```
## 性质
[[联氨的性质]]
### 化学性质
- 是二元弱碱 [[联氨为什么有碱性|why]]（因为有孤对电子）
	- 两步电离
		- $\ce{N2H4 + H2O <=> [N2H5]+ + OH-}$
		- $\ce{N2H5 + H2O <=> [N2H6]+ + OH-}$
- 和酸反应
	- 和少量酸反应
		- →N2H5Cl
	- 和过量酸反应
		- →N2H6Cl2
- 体现还原性（-2价的 N）
	- 作火箭推进剂的原料，和氧气反应
	- →N2+H2O
# 优缺点
- 它们作还原剂的好处：
	- 没有污染（产生 N2）
- 缺点
	- 贵
