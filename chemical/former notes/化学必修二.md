# 化学电池
2. 电池1:正极Cu，负极Zn，电解液硫酸铜
    Zn和硫酸铜反应，Zn失去电子，电子通过外部电线进入正极，到电解液中，把电解液中的铜离子还原为铜（类比把电解液中的银离子还原为银），沉淀在正极上，正极本身的铜没有变化（所以一次性电池正极都是本身不变的，正极一般是石墨）
3. 电池2：正极Ag，负极Fe，电解液$FeCl_3$
    负极的铁和氯化铁反应失去电子，通过外部电路流到正极，进入电解液中，使铁离子被还原为亚铁离子
    正极：$2Fe^{3+} + 2e^- \rightarrow 2Fe^{2+}$负极$Fe - 2e^- \rightarrow Fe^{2+}$
4. 电池3:正极Al，负极Cu，电解液HNO₃
    浓硝酸可以和铝钝化后不发生反应，而铜可以和硝酸发生反应
5. 电池4:正极：石墨，负极Fe，电解液CuCl2
    正极上会有铜析出沉淀在正极上，负极会有Fe溶解，而不是负极上会有铜析出。当nmol个电子通过时，两棒的质量差：（56+64）÷**2**（两个电子）=
6. 电池5:正极Cu，负极Fe，电解液饱和氯化钠
    正极上$O_2$得电子，氯离子移向负极
1. $2FeCl_3 + Cu \rightarrow 2FeCl_2 + CuCl_2$为原理的原电池，正极是碳，负极是铜，电解液为氯化铜。
2. 由于Mg比Al更容易失去电子，所以Mg为负极，Al为正极，电解液可以为$H_2SO_4$
3. 石墨和碳的其他同素异形体可以作为正极，但是金刚石不可以，因为金刚石不导电
4. 正极为铁，负极为Zn，硫酸为电解液的化学电池中，铁不参与反应，只做电子的接收器。
5. 并不是所有原电池负极活动性都比正极材料活动性强（比如锂氟电池），负极不一定失去电子
###
1. 自发的放热化学反应不一定可以设计为原电池（比如氧气和一氧化碳生成二氧化碳），比如有氧气参与点燃的反应、有高温的反应不适合（除了燃料电池外）
2. 电池放电时，化学能不能完全转化为电能，部分转化为热能
3. 燃料电池并没有燃烧
4. 无论是吸热反应还是放热反应，温度上升，反应速度一定增大
5. 铁和硫酸反应时，硫酸浓度增大，氢气产生速率不一定越大（可能钝化等）
6. 当有活泼金属作为电极时，不能用水作为电解液，有氧气也不行，因为会反应
7. 质子交换膜：使负极氢离子通过到正极，并阻止电子通过，从而外部电流有电子
8. 阳离子交换膜，只允许阳离子通过。阴离子交换膜，只允许阴离子通过
9. 化学电池在碱性条件下，碳原子反应会变成$CO_3^{2-}$
10. 化学电池在碱性条件下，Al会变成四羟基合铝酸根
11. 化学电池正极的电势高于负极
12. 原电池氧化还原反应分别在两极，使溶液中粒子运动相互干扰减少，反应更快
2. 有Cu参与的反应中，把Cu换成含有石墨的粗Cu，形成原电池，加快反应速度
3. Zn和 $H_2SO_4$反应中加硫酸铜，由于产生原电池，加快氢气产生速率。如果加过量硫酸铜，Cu会覆盖在Zn表面，减小Zn与$H_2SO_4$接触面积，减小反应速率
4. 阴离子会移动到负极，补偿正电荷的积累
5. 电子走导线，**回到电解质中的是离子**
6. 乙醇是电解质，所以不能作为电解液
7. 原电池中电流表偏向的是正极
# 反应速率
1. 反应中加压使v正和v逆都加快
2. $2KMnO_4 + 5 H_2C_2O_4 + 3 H_2SO_4 \rightarrow K_2SO_4 + 2 MnSO_4 + 10 CO_2 + 8 H_2O$反应中速率不断加快，因为可能生成的硫酸锰是催化剂
3. 证明某反应有反应限度（可逆性）：先证明反应物中还有某物质，证明反应物没反应完；再证明生成物中还有某物质，证明生成物没反应完
4. 反应速率是指一秒钟某物质的**浓度变化**，是平均速度
5. 合成氨工业中，液化氨气并分离，会减小化学反应速率
